Latest Projects

KRichEditor

A rich text editor (based on QuillJs and MRichTextEditor) ported to Kotlin

New

This is a thesis project. This project is an open-source text editor Android library written in Kotlin programming language. More precisely, this would be a bridge between a well-established text editor library written in Javascript named QuillJs and Android API. The user interface is based on another project named "MRichEditor" and the name of this library is "KRichEditor"

"KRichEditor" is written in Kotlin programming language, introduced by Jetbrains, within the famous Android framework provided by Google. Certain popular techniques among Kotlin - Android community such as Kotlin DSLs, functional programming with lambda code blocks, etc. would be implemented. Furthermore, this project also tries to utilize the Kotlin Anko DSL library, which makes the Android development much simpler. All usage methods are also considerable for research purpose. In the end, the compiled artifacts would be hosted at JFrog Bintray, Jitpack and changes are checked through Travis Ci.
